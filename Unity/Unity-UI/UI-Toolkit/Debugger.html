Date: 17/11/2021


Location

Unityâ€™s UI Debugger is a tool that can be used to debug your UI layout to help you find errors in your code.
To find the Debugger Window -> Ui Toolkit -> Debugger & Event Debugger.

Debugger

The Debugger has a toolbar with a dropdown and 2 buttons, the drop down will select root elements, the first button is pick element when selected you can hover over elements and it will update UI Toolkit Debugger, the second button is show layout when selected will apply a box around each element inside of the root element to show the layout of the UI.

3 section the left is the hierarchy, top-right has the same tool as the web margin-border-padding rectangle UXML Dump classes, bottom-right has styles that can be change to the UI Element.
All changes are temporary and when refresh/ deselected it will revert back.

Event Debugger

The Event Debugger has the same dropdown as the Debugger and a Search Field that list all of the Event types
When a root element is selected it will show all selected events that are recorded and displayed in the events log each event has a time stamp the event type and the target of the event.
When you select an event, it will show the event callbacks time stamp handler phase duration
Event propagation paths, event information, registered event callbacks & event durations
